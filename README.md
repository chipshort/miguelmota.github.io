# About

This is my [jekyll](https://github.com/mojombo/jekyll/) powered blog.

[http://miguelmota.github.io/](http://miguelmota.github.io/)

## Instructions

Install deps

```bash
npm install
bundle install
```

Watch and compile

```bash
grunt
```

Compile Sass

```bash
grunt compile-styles
```

Compile scripts

```bash
grunt compile-scripts
```
