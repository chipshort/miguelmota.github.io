mtippy demo

hover over me

Installation (include this in the head section):

  • <script src="jquery.js"></script>
  • <script src="jquery.mtippy.js"></script>
  • <script>
$(document).ready(function(){

	$("[title]").mtippy({

		/* default settings (optional) */
		backgroundColor: "#000",
		borderRadius: ".5em",
		boxShadow: ".1em .1em .5em #000",
		color: "#fff",
		fontFamily: "Helvetica, Arial, sans-serif",
		fontSize: "12px",
		fontStyle: "normal",
		fontWeight: "normal",
		opacity: 1,
		textShadow: "0 -1px 0 rgba(0,0,0,.8)",
		showSpeed: 400,
		hideSpeed: 200,
		timeout: 600

	});
});
</script>
Note: the selector can be anything you want, I just used the title attribute selector for the example
