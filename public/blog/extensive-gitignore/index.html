Extensive .gitignore

April 4, 2013

There are .gitignore templates> suited for pretty much any project. I’m going to share the .gitignore that I recently been using for my web projects. It’s and extended verison of github’s global .gitignore>.

Here is the gist>:

//this will affect all the git repos
git config --global core.excludesfile ~/.gitignore


//update files since .ignore won't if already tracked
git rm --cached <file>
